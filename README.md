# tetris

## 概要
ChatGPTを用いて作成したウェブアプリのテトリスです。

## スクリーンショット
![スクショ](screenshot.gif)

## 操作方法
1. スタートボタンまたはスペースキーを押してゲームを開始
2. 左右矢印キーで落ちているブロックを横移動
3. 下矢印キーで素早く落下
4. 上矢印キーでブロックの向きを変更
5. ゲームオーバー後にリトライボタンまたはスペースキーでリトライ
